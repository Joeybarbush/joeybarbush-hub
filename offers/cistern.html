Tier: Archival · One-shot  ·  contact for rate
// what this is

Some failures are load-bearing. CISTERN is a structured archive of a failure — a decision that went wrong, a build that did not ship, a relationship that ended badly, a period of drift. Joey reads what you send, asks clarifying questions, and writes the archive: a document that holds the loss without sentimentality.

// how it works

The archive is not a postmortem. It is a field document. Something you can return to and find the signal still there.
